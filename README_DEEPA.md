# To Install UV Package Manager 

More Detailed Documentation on UV: https://github.com/astral-sh/uv
- run `curl -LsSf https://astral.sh/uv/install.sh | sh`
- run `uv --version`
- run `uv sync`
Unlike NPM this package manager is so quick.
